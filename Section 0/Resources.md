# 📌 Preparation Tips

- Study consistently every day.
- Focus on concepts before shortcuts.
- Solve previous-year questions regularly.
- Take at least one mock test each week.
- Analyze mistakes after every test.
- Revise weak topics frequently.
- Practice coding by writing code yourself instead of only reading solutions.
- Maintain short notes for quick revision.
