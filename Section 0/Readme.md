# 📝 Assessment Overview

The TCS Ignite assessment is conducted through **TCS iON** and consists of four sections:

| Section             | Focus Area                                     |
| ------------------- | ---------------------------------------------- |
| Numerical Ability   | Quantitative aptitude and problem solving      |
| Verbal Ability      | English grammar, vocabulary, and comprehension |
| Reasoning Ability   | Logical, analytical, and critical reasoning    |
| Coding *(Optional)* | Programming and problem-solving skills         |

> **Tip:** Even though the coding section is optional in the assessment, attempting it can strengthen your overall profile if you're comfortable with programming.
